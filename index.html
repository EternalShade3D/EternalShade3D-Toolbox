📦 Installation Guide

Get up and running in under 2 minutes

Requirements

  • Blender 4.2.0 or later
  • Windows, macOS, or Linux

Steps

  1. Download the addon from Gumroad or FairMarket
  2. Open Blender → Edit → Preferences → Add-ons
  3. Click Install... → select the downloaded .zip file
  4. Enable the checkbox next to "Eternal Utility Toolbox ◈"
  5. Press N in the 3D Viewport → open the ETERNAL tab
First Launch: The addon creates a preferences file in your Blender config directory. You can customize this path in addon preferences (gear icon in the panel header).

📐 Dimensions Guide

See the truth behind your modifiers

What Are "Truth" Dimensions?

Blender's native dimensions property shows the bounding box of the object's cage (original mesh), not what you actually see. When you have modifiers like Subdivision Surface, Solidify, or Geometry Nodes, the visual size differs from the cage size.

Truth Dimensions evaluates the actual result of your entire modifier stack and shows you the real-world size of what you see.

Panel Features

Feature Description
Dimension Sliders (X/Y/Z) Type or drag to set absolute dimensions. Objects scale proportionally.
Uniform Scale When enabled, all three axes scale together — maintains proportions.
Truth Mode Toggle Switch between evaluated "Truth" math and raw "Cage" fallback for selections.
Pivot Point Selector Respects Blender's transform pivot: 3D Cursor, Median Point, Active Element, etc.
Group Bounds Pinning Pin the live bounding box of 2+ selected objects and monitor across selection changes.
Apply Scale Safely applies object scale — handles multi-user mesh data.
Snap to Round Values Rounds dimensions to the nearest 5cm increment.

Edit Mode Selection Dimensions

When in Edit Mode with geometry selected, an additional Local Dimensions section appears. This shows the footprint of your current selection — and when Truth Mode is active, it passes through the modifier stack to show the evaluated size of just the selected geometry.

Smart Sync & Performance

For dense meshes (above the threshold in preferences), the addon uses debouncing to avoid lag. Dimensions update in real-time for light meshes and intelligently cache for heavy ones. The Shield Active indicator tells you when caching is engaged.

Tip: Adjust the Real-time Threshold in Addon Preferences to control when debouncing kicks in. Default is 100,000 vertices.

🎯 Smart Origin Alignment

Snap origins with mathematical precision

Geometry Median

Snaps the origin to the geometric center (mean) of all vertices. Fast and accurate for any mesh density.

Cursor Position

Snaps the origin to the 3D cursor's world location.

9-Point Compass Grid

A 3×3 compass grid lets you snap the origin to any combination of:

  • Top / Center / Bottom (Z-axis)
  • Left / Center / Right (Y-axis)
  • Front / Center / Back (X-axis)
↖ ↑ ↗ (Top row: min/center/max X, max Y) ← • → (Middle: min/center/max X, center Y) ↙ ↓ ↘ (Bottom: min/center/max X, min Y)

Selection-Based Origin

In Edit Mode, snap the origin to the center of your current vertex/edge/face selection.

How It Works: For bounding box calculations on heavy meshes, the addon uses a Geometry Nodes trick — temporarily creating a GN modifier that outputs only the 8 bounding box vertices. This makes even 10M+ vertex meshes compute instantly, then the modifier is destroyed cleanly.

❓ FAQ

Common questions and answers

Why do the dimensions differ from Blender's native display?

Blender shows the cage dimensions (original mesh). This addon evaluates the modified mesh through the entire modifier stack, giving you the real-world size of what you actually see.

The dimensions aren't updating instantly on heavy meshes. Is it broken?

No — this is intentional. For meshes above the Real-time Threshold (default: 100,000 verts), the addon debounces updates to prevent lag. Wait ~0.25 seconds after changes for the sync to kick in. You can adjust the threshold in preferences.

Can I use this in Blender 4.1 or earlier?

No. The addon requires Blender 4.2.0+ due to the new extension system (blender_manifest.toml) and API changes.

Does it work with Geometry Nodes?

Yes! Truth Dimensions evaluates the full GN output, giving you the actual size of whatever your node tree generates.

Where are my naming presets saved?

By default, in your Blender config directory as eternal_naming_data.json. You can change this path in addon preferences.

Is my purchase tied to one machine?

Check the license terms on your purchase platform. Generally, personal use on multiple machines is fine — redistribution is not permitted.
